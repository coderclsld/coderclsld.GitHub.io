## 帖子详情内容
### 一：接口描述

`用于获取帖子详情内容`

### 二：接口地址

**正服**：

http://community.api.66shouyou.com/ll_game_post/api/activity/miniProgram/getGamePostDetail.php

**请求方式**：GET

### 三：请求参数

| **参数** | **名称** | **备注** |
| -------- | -------- | -------- |
| sid      | 帖子id   | 必须     |
|          |          |          |
|          |          |          |

http://community.api.66shouyou.com/ll_game_post/api/activity/miniProgram/getGamePostDetail.php?sid=373392
